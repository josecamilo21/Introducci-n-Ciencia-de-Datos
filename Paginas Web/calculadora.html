Análisis Estadístico de Cacao

Ingeniería y Optimización de Procesos

1. Enunciado del Proyecto

Objetivo: Evaluar la eficiencia del proceso de secado de granos de cacao mediante un modelo de regresión lineal simple.

Descripción: Se han analizado 10 muestras de cacao registrando su Humedad Inicial (X) y su Humedad Final (Y). Buscamos determinar si la humedad inicial es un predictor fiable de la humedad final tras el secado. Este análisis es vital para estandarizar la calidad del cacao exportable.

2. Análisis Inicial en Excel

Antes del análisis avanzado, procesamos los datos en Excel para obtener las métricas base fundamentales:

  • Cálculo de Medias: Determinamos el promedio de humedad inicial (7.42%) y final (5.10%) para entender el punto central de nuestra muestra.
  • Sumatorias de Cuadrados: Excel nos ayuda a calcular las desviaciones de cada punto respecto a la media, base necesaria para hallar la pendiente (m).
  • Coeficiente de Correlación: Identificamos la fuerza de la relación lineal entre las variables de forma rápida.

"El uso de Excel nos permite una visualización tabular rápida, pero para validar si estos datos son científicamente aceptables, debemos recurrir a la potencia de Python."

3. Implementación y Validación en Python

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm
import scipy.stats as stats

# Datos EXACTOS (Humedad Inicial vs Final)
X = [7.42, 7.37, 7.50, 7.55, 7.41, 7.15, 7.44, 7.53, 7.48, 7.39]
Y = [5.03, 5.23, 5.25, 5.10, 5.18, 4.90, 5.08, 5.21, 5.15, 5.05]

# 1. Ajuste del modelo OLS
X_const = sm.add_constant(X) 
modelo = sm.OLS(Y, X_const).fit()

# 2. Resumen Estadístico
print(modelo.summary())

# 3. Diagnóstico de Residuos
residuos = modelo.resid
fig, ax = plt.subplots(1, 2, figsize=(10, 4))
ax[0].hist(residuos, bins=6, color='#5d4037', edgecolor='white')
ax[0].set_title('Distribución de Errores')
stats.probplot(residuos, dist="norm", plot=ax[1])
ax[1].set_title('Gráfico Q-Q de Normalidad')
plt.tight_layout()
plt.show()

¿Por qué este paso es necesario?

Mientras que Excel nos da la recta, Python nos da la certeza:

  • Histograma: Verifica que los errores no tengan tendencias extrañas.
  • Gráfico Q-Q: Si los puntos siguen la línea, el modelo es válido para tomar decisiones industriales.

4. Conclusión Final

Tras el análisis de las 10 muestras, se concluye que existe una relación positiva entre la humedad inicial y la final. El modelo de regresión lineal simple permite predecir el comportamiento del secado con un margen de error controlado.

Impacto: Esta herramienta permite a la planta ajustar los tiempos de secado de forma inteligente, ahorrando energía y asegurando que el cacao no pierda sus propiedades organolépticas por exceso de calor o se dañe por humedad residual excesiva.
