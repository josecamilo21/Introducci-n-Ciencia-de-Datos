Elección del Ejercicio

Suponga una población finita consistente en los valores {0, 2, 4, 6, 8}. Se desea realizar un muestreo aleatorio con reemplazo de tamaño n=2.

Objetivo: Demostrar que la distribución muestral de las medias tiende a una forma acampanada (normal), a pesar de que la población original es uniforme.

Datos de la población:

  • Valores (X): 0, 2, 4, 6, 8
  • Frecuencia: Cada valor tiene una probabilidad de 1/5 (0.2).

Teoría Relacionada

El proceso se basa en dos pilares de la estadística inferencial:

  • Muestreo Aleatorio: Proceso de elección donde todos los individuos tienen la misma probabilidad de ser seleccionados.
  • Teorema Central del Límite (TCL): Establece que, independientemente de la forma de la población original, la distribución de las medias muestrales se aproximará a una normal a medida que n aumenta.

Aunque el TCL suele aplicarse con n ≥ 30, en este ejercicio veremos cómo la forma empieza a cambiar incluso con n=2.

Desarrollo Estadístico

1. Parámetros Poblacionales:

  • Media (μ): (0+2+4+6+8)/5 = 4
  • Varianza (σ²): 8

2. Muestreo (n=2): Al ser con reemplazo, existen 5² = 25 combinaciones posibles.

Código en Python

Simulación profesional utilizando numpy y matplotlib:

import numpy as np
import matplotlib.pyplot as plt
import random

# 1. Definir la población
poblacion = [0, 2, 4, 6, 8]

# 2. Configuración
n = 30  # Tamaño para TCL pleno
num_muestras = 10000
medias_muestrales = []

for _ in range(num_muestras):
    muestra = [random.choice(poblacion) for _ in range(n)]
    medias_muestrales.append(np.mean(muestra))

# 3. Visualización
plt.hist(medias_muestrales, bins=30, color='skyblue', edgecolor='black')
plt.title("Distribución de Medias Muestrales (TCL)")
plt.show()

Análisis Final

ObservaciónResultado
Cambio de FormaDe Uniforme a Campana (Normal).
Validación MediaLa media de las medias es 4.0 (Igual a μ).
DispersiónEl error estándar se reduce según σ/√n.

Conclusión: No necesitamos que la población sea normal para realizar inferencias sobre su media, siempre que el tamaño de muestra sea adecuado.
