Optimización del Secado de Cacao

Análisis Predictivo de Humedad mediante Regresión Lineal

0. Planteamiento del Problema y Estrategia

El Problema: En el procesamiento post-cosecha del cacao, alcanzar una humedad final óptima (cercana al 5-7%) es vital para evitar el moho y garantizar el sabor. Actualmente, existe incertidumbre sobre cómo la humedad de entrada afecta el resultado tras un tiempo fijo de secado.

Objetivo: Construir un modelo matemático predictivo utilizando 10 muestras experimentales para estandarizar el proceso.

Estrategia de Resolución:

  • Excel: Limpieza de datos, cálculo de sumatorias y obtención rápida de la recta de regresión.
  • Python: Validación estadística rigurosa, análisis de residuos (normalidad) y visualización avanzada.

Antes del análisis avanzado, procesamos los datos en Excel para obtener métricas base:

  • Cálculo de Medias: Promedio X: 7.427% | Promedio Y: 5.108%.
  • Sumatorias de Cuadrados: Se calcularon $X^2$ y $XY$ para alimentar las fórmulas de Mínimos Cuadrados.
  • Coeficiente de Correlación (r): 0.6733, indicando una relación positiva moderada.

Bondad de Ajuste ($R^2$): 0.4534 (El 45% de la humedad final se explica por la inicial).

3. Validación y Diagnóstico en Python

Utilizamos Python para verificar que el modelo no sea fruto del azar mediante el diagnóstico de residuos.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm
import scipy.stats as stats

# Datos EXACTOS (Humedad Inicial vs Final)
X = [7.42, 7.37, 7.50, 7.55, 7.41, 7.15, 7.44, 7.53, 7.48, 7.39]
Y = [5.03, 5.23, 5.25, 5.10, 5.18, 4.90, 5.08, 5.21, 5.15, 5.05]

# 1. Ajuste del modelo OLS
X_const = sm.add_constant(X) 
modelo = sm.OLS(Y, X_const).fit()

# 2. Resumen Estadístico
print(modelo.summary())

# 3. Diagnóstico de Residuos
residuos = modelo.resid
fig, ax = plt.subplots(1, 2, figsize=(10, 4))
ax[0].hist(residuos, bins=6, color='#5d4037', edgecolor='white')
ax[0].set_title('Distribución de Errores')
stats.probplot(residuos, dist="norm", plot=ax[1])
ax[1].set_title('Gráfico Q-Q de Normalidad')
plt.tight_layout()
plt.show()
¿Por qué este paso es necesario?

El histograma de errores confirma que los fallos del modelo se distribuyen de forma aleatoria, y el gráfico Q-Q muestra que los puntos siguen la línea ideal, validando científicamente nuestras predicciones para uso industrial.

4. Conclusión Final

Existe una relación positiva entre la humedad inicial y la final. El modelo permite a la planta ajustar los tiempos de secado de forma inteligente, ahorrando energía y asegurando que el cacao no pierda sus propiedades organolépticas.
