# Introducción a Ciencia de Datos 📊

Este repositorio contiene los materiales, notebooks y conjuntos de datos trabajados en la materia **Introducción a Ciencia de Datos**. Aquí se documenta el progreso en el aprendizaje de herramientas para análisis estadístico y manipulación de datos.
